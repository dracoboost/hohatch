# 主な機能

HoHatchは、メイン画面と設定画面の2つの主要な画面で構成されている。

## メイン画面

メイン画面では、画像の変換と管理が可能である。

* **ダンプDDS画像**: [Special K](https://www.special-k.info/) を用いてゲームから抽出されたオリジナルのテクスチャ。
* **インジェクトDDS画像**: シャドウバースWBにインジェクトするテクスチャ。(`XXXXXXXX.dds` のようなファイル名により注入先が定まっている)

DDS画像にカーソルを合わせると、以下のアクションボタンが表示される：

* **JPG → DDS**: 編集したJPGをゲームが必要とするDDS形式に変換し、自動的に `inject/textures` フォルダに配置される。
* **DDS → JPG**: ダンプされたDDSファイルを編集可能なJPG画像に変換する。

### 一括操作

画像名 (`XXXXXXXX` の部分) をクリックすると、複数のファイルを選択できる。複数のファイルが選択されると、以下のボタンが表示される：

* **保存**: 選択した画像をJPGとして、指定したフォルダに一括エクスポートする。
* **削除**: 選択した画像を一括で削除する。
  :::danger[削除について]
  元に戻す機能はないため、削除する際は注意。
  :::

## 設定画面（歯車アイコンからアクセス）

上から順に：

* **言語**: 日本語または英語を選択できます。
* **Special Kフォルダパス**: Special Kの実行ファイルがあるフォルダを指定します。
* **保存するJPGのサイズ**: カード画像を53:64のアスペクト比で保存するオプション (カードの場合はこのアスペクト比が推奨)。
* **システムフォルダ**: キャッシュフォルダとログフォルダを素早く開くためのボタン。
  :::info
  トラブルシューティングに役立つ。問題が発生した場合は、[Discord (HoHatch)](https://discord.gg/fEUMrTGb23) または [X (@dracoboost)](https://x.com/dracoboost) に連絡してほしい。
  :::

<details>
  <summary>エクスプローラーでDDS画像のプレビュー</summary>

  エクスプローラーでDDS画像のプレビューが表示されない場合、[paint.net](https://www.getpaint.net/) のようなDDS互換の画像編集ソフトをインストールすると、プレビューが表示されるようになる。
</details>
