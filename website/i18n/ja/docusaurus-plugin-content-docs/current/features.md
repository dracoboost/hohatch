# 主な機能

HoHatchは、メイン画面と設定画面の2つの主要な画面で構成されている。

## メイン画面

メイン画面では、画像の変換と管理が可能である。

* **ダンプDDS画像**: [Special K](https://www.special-k.info/) を用いてゲームから抽出されたオリジナルのテクスチャ。
* **インジェクトDDS画像**: ビヨンドにインジェクトするテクスチャ。（`XXXXXXXX.dds` のようなファイル名により注入先が定まっている）

DDS画像にカーソルを合わせると、以下のアクションボタンが表示されます：

* **JPG → DDS**: 編集したJPGをゲームが必要とするDDS形式に変換し、自動的に `inject/textures` フォルダに配置します。
* **DDS → JPG**: ダンプされたDDSファイルを編集可能なJPG画像に変換し、お好みの画像エディタで開けるようにします。

### 一括操作

画像名（`XXXXXXXX` の部分）をクリックすると、複数のファイルを選択できます。複数のファイルが選択されると、以下のボタンが表示されます：

* **保存**: 選択した画像をJPGとして、指定したフォルダに一括エクスポートします。
* **削除**: 選択した画像を一括で削除します。
    *元に戻す機能はないため、削除する際はご注意ください。*

## 設定画面（歯車アイコンからアクセス）

上から順に：

* **言語**: 日本語または英語を選択できます。
* **Special Kフォルダパス**: Special Kの実行ファイルがあるフォルダを指定します。
* **保存するJPGのサイズ**: カード画像を53:64のアスペクト比で保存するオプション（カードに推奨）。
* **システムフォルダ**: キャッシュフォルダとログフォルダを素早く開くためのボタン。
    *トラブルシューティングに役立ちます — 問題が発生した場合は、[Discord (HoHatch)](https://discord.gg/fEUMrTGb23) または [X (@dracoboost)](https://x.com/dracoboost) を通じてお気軽に送ってください*

## ヒント

エクスプローラーでDDS画像のサムネイルやプレビューが表示されない場合、[paint.net](https://www.getpaint.net/) のようなDDS互換の画像ビューア/エディタをインストールすると、通常は問題が解決します。
