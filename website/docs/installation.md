# Installation

To use HoHatch, download the latest version of HoHatch from GitHub.  
Extract the downloaded zip file using [7-Zip](https://www.7-zip.org) or similar software, then double-click the HoHatch.exe file inside the folder to launch it.

- **[Download latest HoHatch (direct link)](https://github.com/dracoboost/hohatch/releases/latest)**

:::note[Older versions]
For older versions, visit the [Releases page](https://github.com/dracoboost/hohatch/releases).
:::
