## For Contributors

HoHatch is currently a solo project. Most of the core features I personally needed have already been implemented, but if you would like to add new features and are comfortable writing **Python** and/or **Next.js**, contributions are very welcome!

[GitHub (dracoboost/HoHatch)](https://github.com/dracoboost/hohatch)

Please make sure the preflight check passes before submitting updates.
For detailed guidelines (in English), please read `GEMINI.md`.

```sh
# From the frontend directory
cd frontend
npm run preflight
```
